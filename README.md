#  Jeu du Snake - Projet Python

##  À propos du projet
Ce projet est un clone du célèbre jeu classique "Snake", développé en Python. 
Il a été réalisé de manière autonome afin de mettre en pratique les concepts de **Programmation Orientée Objet (POO)** et la gestion d'interfaces graphiques 2D.

##  Technologies utilisées
* **Langage :** Python 3
* **Bibliothèque :** Pygame
* **Concepts mis en œuvre :** Programmation Orientée Objet, Algorithmique (grille de coordonnées), Gestion des événements et des collisions, Boucle de jeu (Game Loop).

##  Fonctionnalités
* Déplacement continu du serpent sur la grille.
* Génération aléatoire de la nourriture (pommes) avec gestion des bordures.
* Système de croissance du serpent à chaque pomme mangée.
* Architecture propre séparée en plusieurs classes (`Snake`, `Apple`).

##  Installation et Exécution
1. Cloner ce dépôt : 
   `git clone https://github.com/LoukaAyariPignel/SnakeProject.git`
2. Installer les dépendances requises : 
   `pip install pygame`
3. Lancer le script principal : 
   `python main.py`
